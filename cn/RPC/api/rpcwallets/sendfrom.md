# sendfrom 方法

从指定地址，向指定地址转账。

> - 执行此命令前需要 RPC 调用 openwallet 方法来打开钱包
> - 此方法由插件提供，需要安装 [RpcServer](https://github.com/neo-project/neo-modules/releases) 插件才可以调用



```json
{
  "jsonrpc": "2.0",
  "method": "sendfrom",
  "params": [asset_id, from, to, value],
  "id": 1
}
```



### 参数说明

* asset_id：资产 ID（资产标识符），即NEP-5合约的scripthash
* from：转账地址。
* to: 收款地址。
* value：转账金额。
